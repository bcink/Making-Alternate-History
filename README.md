# Making Alternate History

Lander Pack
Update 1.2

Based on the alternate history universe of Eyes Turned Skyward


Thank you for downloading Making Alternate History! I really hope you enjoy this mod! It was a great pleasure to create this pack as 
it received much support from the community as well as many helpful tips and ideas along the way.


Making Alternate History is 100% dependent upon Textures Unlimited by Shadowmage, specifically for the inflatable habitat units.
Please download and install Textures Unlimited before attempting to load the pack.

Textures Unlimited forum link:
https://forum.kerbalspaceprogram.com/index.php?/topic/167450-131-textures-unlimited-pbr-shader-texture-and-model-api-11-11-17/



Installation:

Extract the contents of the GameData folder to your GameData folder. Structure should then read GameData\MAH\


Compatibility patches automatically detect addon based on installed addons.

Patches include:

Connected Living Space
Kerbal Inventory / Attachment System
Snacks
TACLS
Tweakscale
USI Life Support


Pack contains:

Ascent stage
Descent stage
Storage boxes
Ascent engine
Descent engines
RCS thrusters
Antenna
Inflatable habitat
Double-stack habitat
Double-stack inflatable habitat
Solar panels
Ladder
Landing Legs


Parts guide available at https://imgur.com/a/62jEy
